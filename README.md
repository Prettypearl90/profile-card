# profile-card
This is a simple and responsive profile card built using HTML and CSS.

## Features
- Avatar image
- Name and role
- Short bio
- Social media buttons
- Mobile-friendly design

## How to Use
1. Open `profile-card.html` in your browser
2. Customize the content and styles if needed
